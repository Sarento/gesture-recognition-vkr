# Система распознавания Русского Жестового Языка (РЖЯ)

Полная система для распознавания русского жестового языка на основе:
- **MediaPipe Hand Landmarker** - детектирование 21 ключевой точки на каждой руке
- **Slovo Dataset** - открытый датасет РЖЯ (20400 видео, 1000 жестов)
- **ONNX модели** - предобученные модели MViTv2, Swin, ResNet-i3d

## 🚀 Быстрый старт

### 1. Установка зависимостей

```bash
cd gesture-recognition-vkr
pip install -r requirements.txt
```

### 2. Загрузка модели (опционально, но рекомендуется)

Скачайте модель Slovo из папки `models/MODELS_README.md` и поместите в `models/`:

```bash
# Пример для Linux/Mac
wget https://rndml-team-cv.obs.ru-moscow-1.hc.sbercloud.ru/datasets/slovo/models/mvit/onnx/mvit32-2.onnx -O models/mvit32-2.onnx
```

### 3. Запуск системы

```bash
# Режим реального времени с камерой
python run.py

# С указанием модели
python run.py --model models/mvit32-2.onnx

# Только rule-based (без ML модели)
python run.py --no-slovo

# Распознавание на изображении
python run.py --image photo.jpg
```

## 🎯 Поддерживаемые жесты

### Русский алфавит (33 буквы)
А, Б, В, Г, Д, Е, Ё, Ж, З, И, Й, К, Л, М, Н, О, П, Р, С, Т, У, Ф, Х, Ц, Ч, Ш, Щ, Ъ, Ы, Ь, Э, Ю, Я

### Цифры
0, 1, 2, 3, 4, 5

### Базовые слова и фразы
- привет, спасибо, пока
- да, нет, хорошо, плохо
- я, ты, друг, семья
- дом, работа, учеба
- помощь, понимаю, не понимаю

### Полная классификация Slovo (1000 классов)
При использовании ONNX модели поддерживаются все 1000 классов из датасета Slovo:
- Животные (кошка, собака, волк, медведь, и т.д.)
- Прилагательные (добрый, злой, умный, и т.д.)
- Эмоции (радость, грусть, гнев, страх, и т.д.)
- Еда и напитки (есть, пить, ресторан, и т.д.)
- Действия (идти, бежать, спать, и т.д.)
- Предметы (дом, книга, телефон, и т.д.)
- И многие другие категории!

Полный список классов см. в `slovo_repo/constants.py`.

## 📊 Как это работает

### Rule-based распознавание
1. MediaPipe обнаруживает руки и извлекает 21 landmark на каждой руке
2. Вычисляются углы пальцев и их расправленность
3. Сравнивается с базой жестов по формуле:
   ```
   score = 0.7 * finger_match + 0.3 * thumb_score
   ```
4. Применяется временное сглаживание (8 последних кадров)

### ML распознавание (Slovo)
1. Накопление буфера из 16-48 кадров с landmarks
2. Формирование последовательности (1, num_frames, 126)
3. Инференс ONNX модели
4. Получение предсказания из 1000 классов
5. Временное сглаживание результатов

## 🔍 Troubleshooting

### "Модель не найдена"
- Убедитесь, что файл `.onnx` находится в папке `models/`
- Проверьте путь: `python run.py --model models/ваша_модель.onnx`

### "Не удалось открыть камеру"
- Проверьте индекс камеры: `python run.py --camera 1`
- Убедитесь, что камера не используется другим приложением

### Низкая точность распознавания
- Увеличьте освещение
- Держите руки в центре кадра
- Используйте более высокую модель (mvit32-2 вместо mvit16-4)
- Настройте порог уверенности

### Медленная работа
- Уменьшите разрешение камеры
- Используйте `--no-slovo` для отключения ML модели
- Установите `onnxruntime-gpu` для ускорения на GPU

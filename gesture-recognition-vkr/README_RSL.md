# Система распознавания Русского Жестового Языка (РЖЯ)

## Описание

Данная система предназначена для распознавания жестов русского жестового языка в реальном времени с использованием:
- **MediaPipe Hand Landmarker** - для детектирования ключевых точек рук
- **Датасет Slovo** - открытый датасет русского жестового языка
- **Алгоритмы компьютерного зрения** - для классификации жестов

## Возможности

### Поддерживаемые жесты:
1. **Русский алфавит** (А-Я, Ё, Ъ, Ы, Ь)
2. **Цифры** (0-5 и более)
3. **Базовые слова**:
   - привет, спасибо, пока
   - да, нет, хорошо, плохо
   - я, ты, друг, семья, дом
   - работа, учеба, помощь
   - понимаю, не понимаю

### Функции:
- Распознавание статических жестов (положение пальцев, углы)
- Анализ динамических жестов (движения рук)
- Формирование предложений из последовательности жестов
- Временное сглаживание результатов
- Отображение FPS для контроля производительности

## Архитектура системы

### 1. Детектирование рук
Используется MediaPipe Hands для получения 21 ключевой точки на каждой руке.

### 2. Извлечение признаков
- Координаты ключевых точек (x, y, z)
- Углы между фалангами пальцев
- Статус расправленности каждого пальца

### 3. Классификация жестов
Сравнение текущих признаков с базой жестов РЖЯ:
- Совпадение расправленных пальцев (70% веса)
- Совпадение углов большого пальца (30% веса)

### 4. Формирование предложений
- Временное сглаживание предсказаний
- Подтверждение жеста при удержании (>0.5 сек)
- Добавление в предложение без повторов

## Интеграция с датасетом Slovo

Для использования предобученных моделей из датасета Slovo:

1. Скачайте модели с [официального репозитория](https://github.com/hukenovs/slovo)
2. Поместите ONNX модели в папку `models/`
3. Используйте `demo.py` из репозитория Slovo для inference

### Предобученные модели Slovo:
- MViTv2-small (лучшая точность ~64%)
- Swin-large
- ResNet-i3d
- SignFlow-A (SOTA на WLASL-2000)

## Точность распознавания

Точность зависит от:
- Освещения
- Расстояния до камеры
- Угла обзора руки
- Индивидуальных особенностей жестикуляции

Рекомендуется:
- Хорошее освещение
- Камера 720p или выше
- Рука в центре кадра
- Минимум фоновых движений

## Расширение базы жестов

Для добавления новых жестов отредактируйте метод `_create_rsl_gesture_database()` в файле `src/rsl_recognizer.py`:

```python
'новый_жест': {
    'fingers': [0, 1, 0, 0, 0],  # Статус пальцев (0-согнут, 1-расправлен)
    'thumb_angle': 45            # Угол большого пальца (градусы)
}
```

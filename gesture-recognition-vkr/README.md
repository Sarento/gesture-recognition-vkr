# Система распознавания жестов глухонемых людей

## Описание проекта
Данный проект представляет собой систему распознавания жестового языка для перевода слов на русском языке в жесты с использованием MediaPipe Hand Landmarker.

## Структура проекта
```
gesture-recognition-vkr/
├── src/                    # Исходный код
│   ├── __init__.py
│   ├── hand_detector.py    # Детектирование рук
│   ├── gesture_recognizer.py  # Распознавание жестов
│   └── main.py             # Главный файл запуска
├── data/                   # Данные для обучения и тестирования
├── models/                 # Сохраненные модели
├── utils/                  # Вспомогательные функции
│   ├── __init__.py
│   ├── gesture_database.py # База данных жестов
│   └── visualizer.py       # Визуализация результатов
├── tests/                  # Тесты
├── docs/                   # Документация
├── requirements.txt        # Зависимости
└── README.md              # Этот файл
```

## Установка
```bash
pip install -r requirements.txt
```

## Запуск
```bash
python src/main.py
```

## Технологии
- MediaPipe Hand Landmarker - для детектирования ключевых точек руки
- OpenCV - для обработки видео
- NumPy - для вычислений
- scikit-learn - для классификации жестов
